# オフラインセンサーの取り扱い{#dealing-with-offline-sensors}

センサーがオフラインになる2つのシナリオを計画します。

**推奨される頻度：** 必要に応じて

これらのシナリオとは、他のサーバーからWebサーバーがローテーションから取り出され、Webサーバーが失敗によって完全に停止することです。

最初に理解すべき概念は、[!DNL data workbench server]の「基準日」時間です。
