# 棒グラフ{#bar-charts}

棒グラフのビジュアライゼーションを使用すると、各データ要素がグラフ上の横棒として表され、指標およびディメンションデータを視覚化できます。 棒グラフでは、一度に200個までの固有データ要素をグラフに表示できます。 ディメンション値にはy軸に沿ったラベルが付けられ、指標の増分にはx軸に沿ったラベルが付けられます。 各ディメンション値のバーには、正確な指標値のラベルが付けられます。
