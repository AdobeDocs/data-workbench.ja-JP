# 指標の凡例{#metric-legends}

指標の凡例のビジュアライゼーションを使用すると、指標の合計を表形式で表示できます。 最初の列は指標名を保持し、2番目の列は指標値を保持します。 すべての行をウィンドウに一度に表示できない場合は、スクロールバーが表示されます。

特定のディメンション値が表示されていないので、指標の凡例内で選択を行うことはできません。
