# 指標の凡例{#metric-legends}

{{eol}}

指標の凡例のビジュアライゼーションを使用すると、指標の合計を表形式で表示できます。 最初の列には指標名が格納され、2 番目の列には指標値が格納されます。 すべての行をウィンドウに一度に表示できない場合は、スクロールバーが表示されます。

特定のディメンション値が表示されていないので、指標の凡例内で選択を行うことはできません。
