# ワークスペースの共有{#share-a-workspace}

エクスポートワークスペースメニューを使用して、ワークスペースの印刷および .png ファイルまたは Excel へのエクスポートを行えます。

In a workspace, click **[!UICONTROL Export]**.

![](assets/mnu_export.png)
