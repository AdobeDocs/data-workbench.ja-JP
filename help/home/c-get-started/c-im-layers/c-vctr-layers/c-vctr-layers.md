# ベクターレイヤーの操作{#working-with-vector-layers}

{{eol}}

Data Workbenchでは、ベクトルレイヤーは、静的ベクトルデータ（ラインアート）をグローブ上に表示します。

ベクトルレイヤーを使用して、地理的地域（都道府県、国、島、水域など）の輪郭を表示できます。
