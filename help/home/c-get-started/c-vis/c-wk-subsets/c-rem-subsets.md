# サブセットの削除{#remove-a-subset}

データのサブセットを削除する手順です。

* オンラインで作業している間に、ワークスペース内で右クリックし、 **[!UICONTROL Data]** / **[!UICONTROL Subset]** / **[!UICONTROL All]**&#x200B;をクリックします。
