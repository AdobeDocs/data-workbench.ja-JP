# プロセスマップの移動と回転{#move-and-rotate-a-process-map}

ウィンドウ内でのプロセスマップの移動および 3D プロセスマップを回転させる方法に関する情報です。

**ウィンドウ内でプロセスマップを移動するには**

* マップをクリックしながら目的の位置までドラッグします。

**3D プロセスマップを回転させるには**

* マップを右クリックしながらマウスを目的の回転方向に動かして、マップをドラッグします。
