# セグメント {#segments}

セグメントを使用すると、選択範囲から直接ディメンションを作成できます。

セグメントは、ディメンションとして使用したいグループが互いに重ならない別々のグループである場合に、最も役に立ちます。例えば、以前の購入に基づいて、2 つのタイプの買い物客を比較する場合などです。

セグメントを使用しても、影響を受けるのは、そのセグメントディメンションを使用しているワークスペースだけです。
