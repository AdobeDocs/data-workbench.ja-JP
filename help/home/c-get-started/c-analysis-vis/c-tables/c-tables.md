# テーブル{#tables}

テーブルには、1 つ以上のデータのディメンションにまたがる 1 つ以上の指標を表示できます。

次の例に示すテーブルでは、映画ごとの平均スコアが値として表示されるだけでなく、棒の長さでも示されています。
