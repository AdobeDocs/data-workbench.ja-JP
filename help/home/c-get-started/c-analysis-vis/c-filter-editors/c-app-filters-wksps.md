# ワークスペースへのフィルターの適用{#apply-a-filter-to-a-workspace}

ワークスペースにフィルターを適用する手順です。

* フィルターエディターウィンドウで、「**[!UICONTROL Apply Filter]**」をクリックします。

   フィルターエディターウィンドウの枠が黄色に変わり、フィルターがワークスペースに選択を適用したことを示します。
