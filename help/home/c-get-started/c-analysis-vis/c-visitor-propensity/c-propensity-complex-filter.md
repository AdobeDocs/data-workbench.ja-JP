# 複雑なフィルターの説明{#complex-filter-description}

従属変数とトレーニングフィルターに「複雑なフィルターの説明を表示」を追加できます。

**[!UICONTROL Show the Complex Filter Description]**&#x200B;に対して、スコアリングダイアログから[!DNL Options > Show Complex Filter Description]を選択します。
