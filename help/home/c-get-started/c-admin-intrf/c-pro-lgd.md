# 処理の凡例{#processing-legend}

処理の凡例には、特定のサーバーのデータ処理および変換に関する詳細情報が表示されます。この情報を使用して、再処理および再変換中のデータの進行状況を追跡できます。

![](assets/vis_ProcessingLegend.png)

次の表に、[!DNL Processing Legend]を使用して完了できるタスクを示します。

  </tr>
