# 補足的な設定ファイル{#additional-configuration-files}

データセットプロファイル（またはその継承プロファイル）の Dataset ディレクトリに格納されるいくつかの補足的な設定ファイルについて取り上げます。

データセットプロファイルを定義するうえで必須というわけではありませんが、これらのファイルを編集することによって、データ処理のパラメーターを別途指定することができます。
