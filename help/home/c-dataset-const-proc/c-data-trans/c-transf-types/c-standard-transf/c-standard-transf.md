# 標準変換{#standard-transformations}

標準変換は、一連の入力を受け取って何らかの処理を実行した後、その結果を 1 つまたは複数の出力フィールドに格納します。

以下の節には、各標準変換の説明、使用可能なパラメーターの一覧および例が記載されています。Data Workbench サーバーの DPU はいずれも、すべての標準変換に対応しています。
