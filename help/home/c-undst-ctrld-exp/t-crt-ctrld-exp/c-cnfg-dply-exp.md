# 実験の設定とデプロイ{#configuring-and-deploying-the-experiment}

{{eol}}

目標、仮説、実験の詳細を定義し、テストコンテンツを作成したら、対照実験を導入するようにセンサーを設定する必要があります。

## 実験設定ファイルの設定 {#section-037fe7dea9c94aee9cdc354dafdb7c03}

実験を設定するには、Adobeが提供する実験設定スプレッドシート ( [!DNL TestExperiment.xls] （デフォルト）。 このファイルでは [!DNL Sensor] 実験を実行するには、とは、 [ExpFile パラメーターの変更](../../../home/c-undst-ctrld-exp/t-en-ctrld-exp/c-mod-expfile-prm.md#concept-25232b386a654870becc789d4f1fcc28).

このファイルには、複数の実験に関する情報を含めることができます。複数の実験は、同じ時間に、または異なる時間に実行し、異なるグループと割合を使用できますが、これらの実験はどのような相関関係もありません。

ユーザーは、この時点で実行するように設定されたファイル内の実験リストごとに、グループに分類されます。

>[!NOTE]
>
>各実験は、他のすべての実験とは独立しています。 ある実験に加えた変更は、他の実験には影響しません。また、訪問者は複数の実験に属する場合がありますが、結果は互いに関係しません。 複数の実験での変化の間に相関関係があると思われる場合は、これらの変化をテストする新しい実験を作成する必要があります。

**実験を設定するには**

このファイルは、実験の開始前に入力し、実験の実行中に情報を変更しないでください。

>[!NOTE]
>
>実験の開始後に実験の定義が変更されると、実験は即座に無効になります。

1. Web またはアプリケーションサーバーへの管理者アクセス権がある場合は、 [!DNL Sensor] 任意の [!DNL Sensor] web クラスター内のマシンで、にアクセスする [!DNL TestExperiment.xls] ファイル。 管理者アクセス権がない場合は、担当のAdobeアカウントマネージャーに連絡し、 [!DNL TestExperiment.xls] ファイル。

1. を開きます。 [!DNL TestExperiment.xls] ファイル（必要に応じてこのファイルの名前を変更できます）を編集し、次のフィールドに入力します。

## 設定ファイルのデプロイとコンテンツのテスト {#section-34ff29649f584b93bc6129b75084b37c}

実験設定ファイルは、 [!DNL Sensor] 実験に関係するページを提供する 手動の手順または既存のコンテンツ管理システムを使用して実行できます。

**テストコンテンツをデプロイするには**

* を実行する各アプリケーションまたは Web サーバー上 [!DNL Sensor] 実験に関係するページを提供するために、既存の公開プロセスを使用して、テストコンテンツを適切な場所にデプロイします。

   例えば、テストグループページを公開する場合は、 [!DNL index2.asp] を web サイトのテストフォルダー ( [!DNL mysite.com])、ファイルを [!DNL www.mysite.com/test].

   >[!NOTE]
   >
   >Web サイト上のページから直接テストファイルにリンクしないでください。 これにより、テスト結果とインデックススコアが無効化されます。

**実験をデプロイするには**

* を実行する各アプリケーションまたは Web サーバー上 [!DNL Sensor] 実験に関係するページを提供する実験設定テキストファイルを、ExpFile パラメーターで指定したディレクトリ ( [!DNL Sensor] 設定ファイル。 詳しくは、 [ExpFile パラメーターの変更](../../../home/c-undst-ctrld-exp/t-en-ctrld-exp/c-mod-expfile-prm.md#concept-25232b386a654870becc789d4f1fcc28).

[!DNL Sensor] ファイルで定義した割合に基づいて各グループの Web サイトの訪問者がランダムに選択され、必要に応じてテストグループまたはコントロールグループのコンテンツが提供されます。
