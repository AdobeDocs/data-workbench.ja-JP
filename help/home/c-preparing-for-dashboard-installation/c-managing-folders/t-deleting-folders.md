# フォルダの削除{#deleting-folders}

1. 削除するフォルダを右クリックし、を選択しま **[!UICONTROL Delete]**&#x200B;す。

   ![](assets/delete_folder.png)

   >[!NOTE]
   >
   >フォルダを削除する前に、フォルダを空にする必要があります。

1. フォルダを削除するかどうかを確認するプロンプトが表示されます。
1. 操作が正常に完了すると、フォルダーが削除されたことを示すプロンプトも表示されます。
