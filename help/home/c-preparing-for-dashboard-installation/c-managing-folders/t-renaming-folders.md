# フォルダー名の変更{#renaming-folders}

{{eol}}

1. 名前を変更するフォルダを右クリックし、「 」を選択します。 **[!UICONTROL Rename…]**.

   ![](assets/rename.png)

1. フォルダーの新しい名前を入力し、を押します。 **[!UICONTROL Enter]**.

   ![](assets/renamed_folder.png)

   操作が成功した場合は、フォルダーの名前が変更されたことを示すプロンプトが表示されます。
