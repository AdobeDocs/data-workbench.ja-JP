# フォルダ名の変更{#renaming-folders}

1. 名前を変更するフォルダを右クリックし、を選択しま **[!UICONTROL Rename…]**&#x200B;す。

   ![](assets/rename.png)

1. フォルダの新しい名前を入力し、を押しま **[!UICONTROL Enter]**&#x200B;す。

   ![](assets/renamed_folder.png)

   操作が正常に完了すると、フォルダーの名前が変更されたことを示すプロンプトが表示されます。
