# ユーザーアカウントの編集{#editing-a-user-account}

{{eol}}

1. 歯車アイコンをクリックします。 ![](assets/edit_icon.png) を編集するユーザーの行の最後に追加します。

   ![](assets/edit_user_account.png)

1. ユーザーのアカウント設定を編集し、 **[!UICONTROL Save]**.

   操作が成功した場合は、ユーザーグループが正常に保存されたことを示すメッセージが表示されます。
