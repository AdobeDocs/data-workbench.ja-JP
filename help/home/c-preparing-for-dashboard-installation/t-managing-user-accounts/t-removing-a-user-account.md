# ユーザーアカウントの削除{#removing-a-user-account}

{{eol}}

1. 次をクリック： **[!UICONTROL X]** システムから削除するユーザーの行末のアイコン。

   ユーザーを削除するかどうかを尋ねるプロンプトが表示されます。 1. 「 」を選択します。 **[!UICONTROL Yes]** をクリックして、ユーザーを削除します。

   操作が成功した場合は、ユーザーが正常に削除されたことを示すメッセージが表示されます。
