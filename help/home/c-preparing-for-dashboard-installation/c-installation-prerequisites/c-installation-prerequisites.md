# インストールの前提条件{#installation-prerequisites}

インストールを開始する前に、サーバーを設定し、新しいソフトウェアをインストールする権限を持つ管理者アカウントでWebサーバーにログオンします。

>[!NOTE]
>
>ダッシュボードシステムは、1つの層（IISと同じサーバ上のデータベース）または2層のシステムとして展開できます。 このガイドでは、2層アプローチに関する追加の設定手順を説明しません。
