# ユーザーグループの削除{#removing-a-user-group}

1. システムから削除するグループの行の最後にある&#x200B;**[!UICONTROL X]**&#x200B;アイコンをクリックします。

   グループ1を削除するかどうかを尋ねるプロンプトが表示されます。 「**[!UICONTROL Yes]**」をクリックします。

   操作が成功した場合は、グループが正常に削除されたことを示すメッセージが表示されます。
