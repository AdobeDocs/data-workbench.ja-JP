# レポートポータルの使用{#using-the-report-portal}

レポートポータルでは、Report Serverで生成されたレポートをWebブラウザー経由で表示できます。

は、Microsoft [!DNL Report Portal] Internet Information Services (IIS) Webサーバー上で実行されるWebベースのアプリケーションです。
