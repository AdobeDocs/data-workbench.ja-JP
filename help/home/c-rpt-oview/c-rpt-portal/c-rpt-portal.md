# Report Portal の使用{#using-the-report-portal}

Report Portalでは、Report Serverが生成したレポートをWebブラウザーで表示できます。

[!DNL Report Portal]は、Microsoft Internet Information Services(IIS)Webサーバー上で動作するWebベースのアプリケーションです。
